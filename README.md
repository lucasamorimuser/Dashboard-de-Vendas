### Linguagem e Ferramentas

Excel - Armazenamento dos dados em arquivo no formato .CSV

PowerBi - Ferramenta de elaboração e análise gráfica

### Roteiro

Importação dos dados no formato .CSV

Contrução de gráficos:

 - De linha para análise de faturamento de venda de cada produto ao longo do tempo;
 - De donuts para a contribuição percentual de cada produto no faturamento total;
 - De barras horizontais para melhor compreensão do faturamento (em R$) por período do dia;
 - De barras horizontais para ranquear os produtos por faturamento (em R$)

Interatividade com o usuário:

  - Além de dotos os campos "clicáveis" dos gráficos que funcionam como interface interativa, foi adicionado um cartão de linha múltipla em formato de menu que atua como botões de seleção para que seja possivel filtrar os dados de acordo com o produto desejado.

### Análise dos dados

Os produtos Latte e Americano With Milk somam quase 50% do faturamento da loja. Esses produtos merecem atenção pela sua representatividade.

O Café Americano está em uma alta de vendas e esta pode ser explorada para que perdure, já que, é o 4° produto em participação no faturamento da loja.

É necessário avaliar as abordagens de venda do Cortado que está em queda constante desde o período de dez/2024.

O Expresso tem o pior desempenho de vendas. Cabe avaliação de estratégias de venda para fomentar um melhor desempenho, ou então, a retirada da opção do menu abrindo espaço para um novo produto.
